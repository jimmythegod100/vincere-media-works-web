FAQ

Common questions

How do I start a project?

Fill out the contact form below with your project type, timeline, and budget range. We'll reply within 24–48 hours with next steps and a quote.

What's your typical turnaround time?

Starter projects: 3–5 business days. Standard packages: 1–2 weeks. Pro productions: 2–4 weeks depending on scope. Rush delivery available for an additional fee.

Do you offer revisions?

Yes. Every package includes at least one revision round. Additional revisions can be added. We want you 100% satisfied before final delivery.

What file formats do you deliver?

Video: MP4 (H.264), ProRes on request. Design: PNG, SVG, PDF, and source files. Web: live URL plus source code when applicable.

Do you work with clients remotely?

Absolutely. We work with clients nationwide via email, video call, and cloud file sharing. Location is never a barrier.
